# AI-Powered Learning Outcomes vs Job Market Demand Analysis

## 📌 Problem Statement
This project analyzes the relationship between online learning behavior and job market demand to identify whether high-demand fields also lead to higher learning success rates.

---

## 📊 Dataset
- Learning Dataset (1000 records)
- Job Dataset (73,000+ records)

---

## ⚙️ Approach
- Data Cleaning & Preprocessing using Python (Pandas)
- Aggregation & Analysis (GroupBy, Mean, Sorting)
- Created job categories for mapping
- Compared learning outcomes with job demand
- Integrated AI (Google Gemini API) for automated insights

---

## 🔍 Key Insights
- Computer Science & Engineering have highest completion rates
- Data Science has highest job demand but lower completion rate
- High job demand ≠ high learning success

---

## 🤖 AI Integration
Used Google Gemini API to generate insights based on processed data.

---

## 📈 Conclusion
There is a clear gap between job demand and learning success. Foundational technical fields show better alignment, while high-demand fields like Data Science require stronger learning support.
